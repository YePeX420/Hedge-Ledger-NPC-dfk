# DFK PvE Extension Data Bundle

This bundle contains machine-ready starter files for the PvE extension.

Files included:
- `consumables.master.json` — complete validated consumable dataset with weights
- `statuses.master.json` — status behavior blueprint for the simulator
- `ability.schema.json` — canonical normalized ability schema for future `abilities.master.json`

Notes:
- Consumables are complete and validated from screenshots.
- Statuses are a complete blueprint for deterministic simulation logic.
- The full `abilities.master.json` is large and should be generated from the validated class transcriptions using the schema in this bundle.
- The simulator should prefer fetched live combat stats as the source of truth for final combat parameters, rather than reconstructing gear and pet stat contributions unless hidden procs need separate modeling.
